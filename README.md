# MERN Todo Application

## Overview

A modern, responsive, and full-stack Task Management Application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. The application enables users to efficiently create, manage, update, and delete tasks through an intuitive user interface featuring a modern glassmorphism-inspired design.

This project demonstrates end-to-end development of a production-style web application, including frontend development, RESTful API integration, database operations, state management, and responsive design principles.

---

## Key Features

- Create, Read, Update, and Delete (CRUD) tasks
- Responsive user interface for desktop and mobile devices
- MongoDB database integration using Mongoose
- RESTful API architecture
- Modern Glassmorphism UI Design
- Real-time state updates using React Hooks
- Error handling and user feedback notifications
- Clean and maintainable component-based architecture
